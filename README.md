I work at the intersection of **defense and offense** — watching logs for signs of trouble, and probing web apps and APIs for the holes an attacker would find first. Based in Nepal 🇳🇵, building toward a career as a SOC Analyst and penetration tester.

### What I actually do

```
$ whoami
├── Monitor & triage security events (Splunk, Suricata)
├── Hunt for threats before they become incidents
├── Test web apps & APIs against the OWASP Top 10
├── Run vulnerability scans and chase down what they find
└── Harden Linux systems so there's less to find in the first place
```
